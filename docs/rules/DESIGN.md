# Design Rules

## Direction

- 기존 포트폴리오의 이력서형 구조, 컴팩트한 섹션, 작은 카드 반경을 유지한다.
- 색상은 파스텔 톤으로 조정하되 한 가지 색 계열에 치우치지 않는다.
- 라이트/다크 모드는 같은 토큰 이름을 공유하고 `data-theme`로 전환한다.

## Layout

- 첫 화면은 포트폴리오 본문이어야 하며 별도 마케팅형 랜딩 페이지를 만들지 않는다.
- 카드 안에 또 다른 장식 카드를 중첩하지 않는다.
- 모바일과 데스크톱에서 텍스트가 버튼이나 카드 밖으로 넘치지 않게 한다.

## Interaction

- 테마 토글은 우측 하단 고정 버튼으로 제공한다.
- 포커스 가능한 요소는 `:focus-visible` 스타일을 유지한다.
- 모달과 채팅 입력은 키보드 접근성을 유지한다.
