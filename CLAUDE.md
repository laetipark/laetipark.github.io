# CLAUDE.md

이 파일은 Claude Code용 짧은 작업 라우터다.

작업 전 반드시 `docs/README.md`를 먼저 읽는다.
`docs/README.md`는 작업 하네스이자 문서 분류표이며, 필요한 `docs/rules/*.md`를 선택하는 기준이다.

## 실행 순서

1. `docs/README.md`를 읽는다.
2. 작업 성격에 맞는 `docs/rules/*.md`를 선택해 읽는다.
3. 선택한 규칙을 현재 작업의 기준으로 삼아 구현, 수정, 검증을 진행한다.
4. 작업이 여러 범주에 걸치면 관련 세부 문서를 모두 읽는다.

## 운영 원칙

- 실제 규칙 정본은 `docs/`에 둔다.
- 이 파일에는 세부 구현 규칙을 중복 작성하지 않는다.
- 새 규칙이 필요하면 `docs/rules/`에 추가하고 `docs/README.md`의 문서 분류표에 연결한다.
- 규칙이 충돌하면 사용자 요청, 선택한 세부 규칙, `docs/README.md`, 이 파일 순서로 우선한다.
